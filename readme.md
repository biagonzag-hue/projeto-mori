# Introdução

*O aprendizado é uma jornada longa, árdua e igualmente recompensatória.* 

A **neuroplasticidade** é um conceito da neurociência que trata a capacidade do cérebro de se reestruturar e se reorganizar ao longo da vida, criando novas conexões neurais para se adaptar a novas experiências e aprendizados. 
Em outras palavras, é o processo pelo qual o cérebro **aprende**, **se adapta** e **se transforma** de acordo com estímulos, experiências, prática contínua e desafios cognitivos. Essa propriedade explica por que conseguimos adquirir novas habilidades, aprimorar conhecimentos com o estudo e até compensar limitações ou recuperar funções após lesões.

Dentre os assuntos que foram aprofundados em aula estão:

1. Estrutura e Funcionamento da Internet

Inclui backbone, infraestrutura física, topologias de rede e princípios básicos de comunicação.

2. Protocolos e Modelos de Comunicação

Abrange camadas OSI/TCP-IP, principais protocolos (TCP, UDP, HTTP/HTTPS, DNS, DHCP etc.) e suas funções.

3. Endereçamento e Roteamento

Envolve IPv4, IPv6, NAT, máscaras de rede e organização lógica da comunicação entre dispositivos.

4. Segurança e Proteção de Dados

Compreende HTTPS/TLS, VPN, LGPD, boas práticas de privacidade e mecanismos de segurança em redes.

5. Desempenho, Qualidade e Escalabilidade

Trata de QoS, balanceamento de carga, redundância e técnicas para suportar grande volume de usuários.

6. Tecnologias da Web e Evolução da Internet

Inclui Web 1.0/2.0/3.0, AJAX, aplicações modernas, APIs REST e arquiteturas web.

As aulas de Controle de Versão foram extremamente proveitosas e essenciais para compreender como projetos reais são organizados, versionados e colaborativos. Para eu, uma completa leiga, a disciplina foi muito importante, principalmente pelo seu caráter prático, onde cada exercício trouxe situações que ampliaram minha compreensão e reforçaram minha autonomia no uso do Git tanto no terminal quanto em interfaces como o VSCode. Dentre os assuntos que foram aprofundados em aula estão:

1. Fundamentos do Git e Controle de Versão

O que é o Git; Configurações globais e locais; Funcionamento interno (.git, commits, árvore, hash, objetos); 

2. Ciclo de Trabalho no Git

Working Tree, Index (Staging Area) e Último Commit; Comandos essenciais: add, commit, status, log, restore. 

3. Branches e Fluxos de Desenvolvimento

Criação, troca e uso de branches; HEAD e apontamento de commits; Boas práticas de nomeação;

4. Merge, Conflitos e Resolução

Funcionamento do merge; Conflitos e como resolvê-los; Fast-forward vs 3-ways; Pull Requests e revisão de código (Approve, Comment, Request Changes);

5. GitHub e Repositórios Remotos

Criar repositórios remotos; Adicionar remoto, push, pull e fetch; Trabalhar colaborativamente; Fluxo local vs fluxo via PR

6. Ferramentas e Extensões

GitLens; VSCode para staging, commits e diff

7. Colaboração e Trabalho em Equipe**

Adicionar colaboradores; Sincronização entre repositórios local/remoto; Resolução de divergências entre commits; PRs com conflitos
